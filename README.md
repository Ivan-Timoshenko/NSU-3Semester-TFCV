Это затеханная версия лекций Абросимова Н.В. по теории функций комплексного переменного в третьем семестре.
Файл **main.pdf** содержит весь материал, который я успел затехать к текущему моменту. 
Тех одной лекции занимает около 1,5 часов, а значит новая лекция будет сделана за 1,5 часа в течение недели.

Файл "colloq.pdf" содержит в себе расписанные 8 вопросов к зачету у О. А. Лазаревой. Список вопросов:
1. Вторая теорема Абеля.
2. Теорема Коши-Адамара
3. Геометрический смысл модуля и аргумента производной. Определение конформного
отображения.
4. Условия дифференцируемости Коши-Римана. Определение аналитической функции.
5. Риманова поверхность функции e^z, z^n. Обращение этих функций. Понятие многолистной
функции, ветви многолистной функции, точки ветвления и точки отделимости многолистной
функции.
6. ДЛО. Определение и основные свойства (круговое, симметрическое). Конформность ДЛО на
всей комплексной плоскости.
7. Формулы стереографической проекции.
8. Теорема Коши о равенстве нулю интеграла по границе области для функции, аналитической в
данной области.
